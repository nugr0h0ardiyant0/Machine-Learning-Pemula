# Machine-Learning-Pemula
Repositori ini berisi dokumentasi belajar Machine Learning, mencakup:  Dasar &amp; Workflow: Intro ML &amp; alur kerja proyek.  Supervised: Klasifikasi (KNN, Decision Tree) &amp; Regresi Linear.  Unsupervised: Clustering (K-Means).  Optimasi: Feature Engineering, penanganan Over/Underfitting, &amp; Hyperparameter Tuning.  Proyek Akhir: Model ML data tabular.
