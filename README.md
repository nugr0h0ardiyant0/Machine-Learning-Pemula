# 🚀 DBS x Dicoding Coding Camp: Machine Learning for Beginners

Repositori ini berisi dokumentasi, catatan, dan proyek yang dikerjakan selama mengikuti alur belajar **Machine Learning** di program **DBS Foundation Coding Camp**.

## 📚 Daftar Materi Pembelajaran

### **Dasar & Alur Kerja**
* **Hi, Machine Learning:** Pengenalan dasar, definisi, sejarah, serta implementasi ML di berbagai industri.
* **Machine Learning Workflow:** Mempelajari siklus hidup proyek ML mulai dari *data acquisition*, *preprocessing*, hingga *model deployment*.

### **Model Supervised Learning**
* **Klasifikasi:** Implementasi algoritma untuk data kategorikal menggunakan **K-Nearest Neighbors (KNN)** dan **Decision Tree**.
* **Regresi:** Teknik prediksi nilai kontinu menggunakan **Linear Regression** serta cara evaluasinya.

### **Model Unsupervised Learning**
* **Clustering:** Teknik pengelompokan data tanpa label menggunakan algoritma **K-Means** dan metode evaluasinya.

### **Optimasi & Performa Model**
* **Feature Engineering:** Strategi transformasi fitur untuk meningkatkan performa model.
* **Model Analysis:** Memahami konsep **Overfitting** dan **Underfitting** serta cara mengatasinya.
* **Hyperparameter Tuning:** Teknik optimasi parameter model untuk mendapatkan akurasi terbaik.

## 🎓 Proyek Akhir (Submission)
Membangun model Machine Learning (Supervised & Unsupervised) menggunakan data tabular untuk menyelesaikan kasus nyata.
